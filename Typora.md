### Заголовок

```markdown
# Заголовок H1
## Заголовок H2
### Заголовок H3
#### Заголовок H4
##### Заголовок H5
###### Заголовок H6
```

### Текст

```markdown
*Курсив* и _Курсив_
**Жирный текст** и __Жирный текст__
***Жирный и курсивный текст***
~~Зачёркнутый текст.~~
```

### Цитаты

```markdown
> Очень глубокомысленная цитата. Пожалуйста, прочтите её внимательно.
>
> Оскар Уайльд
```

### Маркированные списки

```markdown
- Первый пункт.
- Второй пункт.
- Третий пункт.
```

### Нумерованные списки

```markdown
1. Первый пункт.
2. Второй пункт.
3. Третий пункт.
```

### Смешанный вложенный список

```markdown
1. Первый пункт.
- Первый подпункт.
- Второй подпункт.
- Третий подпункт.
2. Второй пункт.
- Первый подпункт.
- Второй подпункт.
- Третий подпункт.
```
